Project Overview

The Employee Registration System is a simple command-line application designed to manage employee records efficiently. It allows adding, editing, deleting, and viewing employee information, all stored locally without external databases.

Key Features

Add New Employees

Register employees with essential details using an easy-to-use interface.

Edit Employee Data

Update existing records quickly through the command line.

Delete Records

Remove employees from the system securely.

View All Employees

Display a complete list of employees for reference.
